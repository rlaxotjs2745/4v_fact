이용료 산출 기준 및 방법
실증서비스 이용료와 보증금은 실증단지 위탁자(실증단지를 소유한 지방자치단체)가 정한 기준과 방법을 따라 실증센터가 산출합니다.

  • 자율실증의 경우 경영비는 제외되며, 장비임대료·분석수수료 등은 입주자의 필요시 발생할 수 있음
